테스트 페이지 - 모든 기능

이 페이지에서는 애플리케이션의 모든 기능을 테스트할 수 있습니다.

사용자 관리

  • 회원가입
    새로운 사용자 계정을 생성합니다.
  • 로그인
    기존 계정으로 로그인합니다.
  • 사용자 정보
    현재 로그인한 사용자의 정보를 확인합니다.
  • 사용자 정보 수정
    사용자 정보를 수정합니다.
  • 회원 탈퇴
    사용자 계정을 삭제합니다.

레시피

  • 레시피 등록
    새로운 레시피를 등록합니다.
  • 레시피 목록
    등록된 모든 레시피를 확인합니다.

공지사항

  • 공지사항 목록
    모든 공지사항을 확인합니다.

Q&A 게시판

  • Q&A 게시판 목록
    모든 Q&A 게시글을 확인합니다.

리뷰

  • 리뷰 작성 (레시피 ID: 1)
    레시피에 대한 리뷰를 작성합니다.

장바구니 및 결제

  • 장바구니
    장바구니를 확인합니다.
  • 결제
    결제 페이지로 이동합니다.
